## Publicación en GitHub Pages

Este entorno no tiene `gh` instalado ni acceso preparado a tu cuenta de GitHub, así que la web está lista localmente pero no se ha podido publicar aún desde aquí.

Flujo típico:

1. `git init`
2. `git add .`
3. `git commit -m "Pagina dia del padre"`
4. Crear un repositorio en GitHub y añadir el remoto.
5. `git push -u origin main`
6. Activar GitHub Pages sobre la rama principal.
